About the Campus Finder

The Farook College Campus Finder helps students, staff and visitors locate any department, office or facility on an interactive map—fast and easy.

Go to Campus Map

Features

  • Leaflet.js interactive map with smooth fly-to animation
  • Powerful building name search
  • Clickable list with instant zoom
  • Dark mode for night-time viewing
  • Quick dropdown location selector

How to Use

  1. Select a location or click “Go to Campus Map.”
  2. Use the map search or sidebar list to find places.
  3. The map zooms to the selected building.
  4. Tap a marker for building details.
